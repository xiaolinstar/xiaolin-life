# 学习、实践、理解 CI/CD 与 DevOps

> 在计算机领域有很多概念比较抽象或难以理解，比如敏捷开发、领域驱动模型。

相信大家对运维领域 CI/CD DevOps 有一个粗浅的理解，但总觉得不能理解全貌。

因此，从我的开发实践经验来思考与探索，希望与大家一起交流。


## 项目实践

本系列文章从开发工程师的项目实践经历，由浅入深理解 CI/CD 与 DevOps。

文章描述上从实践出发，但不过多介绍具体的执行步骤，适合有一定项目部署经验的前端开发工程师、服务端开发工程师以及运维工程师，也适合想了解 CI/CD 与 DevOps 概念、正尝试将手动部署迁移到自动化部署上的开发者。

描述最流行的两类开发项目作为示例：

1. 开发 Vue3 前端 Web 项目，使用 Vite（Node.js） 构建，并部署到云服务器上
2. 开发 SpringBoot 服务端项目，使用 Maven 构建，并部署到云服务器上
