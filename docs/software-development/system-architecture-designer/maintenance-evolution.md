## 负载均衡

负载均衡技术：

- 基于 http 的负载均衡，应用层实现，重定向。
- 反向代理负载均衡，Nginx
- 基于 DNS 的负载均衡，服务失效，性能瓶颈
- 基于 NAT 的负载均衡


负载均衡算法氛围静态均衡算法，和动态均衡算法。

静态：轮询、加权轮询、随机、IP 哈希

动态：最少连接数、加权最少连接数、会话保持

数据同步方案：触发器、定时器、数据库插件
