# 掼蛋

> 由斗地主和双升演化而来

![鬼牌](/images/img-table-game-guandan/joker.webp)

掼蛋是一款起源于中国江苏的扑克牌游戏，由4人参与采用**对家搭档**的协作对抗模式。游戏通过升级、炸弹与牌型组合，将策略规划、记忆推算与团队配合融为一体，兼具竞技性与社交性，牌局节奏明快，深受大众喜爱。

## 获胜机制

4名玩家围坐方桌，对家为一组搭档，胜负由“升级”判定，目标是率先打过最高级别“A”。

![4名玩家](/images/img-table-game-guandan/4玩家.png)

升级顺序：2 → 3 → 4 → … → 10 → J → Q → K → A。

当一组搭档中有人成为**头游**（最先出完手牌的玩家）时，该组获得升级资格，升级数根据搭档名次判定：

- 次游（二游）：第2名出完手牌，**升3级**
- 三游：第3名出完手牌，**升2级**
- 末游：第4名出完手牌，**升1级**

特别地，当对战级别为“A”时，需达成“翻山”条件才算过关：组内一人为头游，且另一人不能是末游。

**获胜规则约定**

- 规则一：若未满足（如头游搭档为末游），则需继续打“A”，直至翻山成功。
- 规则二：若3次未翻山成功，则对手方获胜。

## 基础牌型

基础牌型与斗地主相似，分为**炸弹**和**常规牌**，炸弹可以压过任意常规牌，同一牌型下大牌压小牌。

所有牌型如下表所示，X Y 表示任意牌值，X $\neq$ Y。

| --------- | ----- | -------------------- | -------- | -------- | -------- |
| 炸弹 ↓   | 王炸  | 4张鬼牌(Joker)       | -        | -        | -        |
|           | 8炸   | 8X                   | 33333333 | AAAAAAAA | 22222222 |
|           | 7炸   | 7X                   | 3333333  | AAAAAAA  | 2222222  |
|           | 6炸   | 6X                   | 333333   | AAAAAA   | 222222   |
|           | 同花顺 | *[X,X+1,X+2,X+3,X+4] | *23456   | *10JQKA  | *A2345   |
|           | 5炸   | 5X                   | 33333    | AAAAA    | 22222    |
|           | 4炸   | 4X                   | 3333     | AAAA     | 2222     |
| 常规牌 → | 三连对 | 2[X,X+1,X+2]         | 334455   | QQKKAA   | AA2233   |

**规则提醒**

- 炸弹：4张鬼牌为最大炸弹，同花顺为介于5炸和6炸间的特殊炸弹
- 临街牌A：A作为“1”最小，否则为最大
- 三带对：只比较三张大小，所带对子不参与比较

## 主牌

当局对战级的牌为**主牌**，主牌共8张，拥有特别权限：

- 大小：介于鬼和A之间，鬼 > 主牌 > A
- **红心**赖子：可变换为鬼牌外的任意牌（值与花色）

![红心牌](/images/img-table-game-guandan/红心J.png)

## 进贡与抗贡

上局获胜情况判定进贡，进贡与还贡牌数为1张。

- 进贡：最大手牌，大王 > 小王 > 主牌 > A ...
- 还贡：不大于10的手牌，2,3,...,10

头游玩家获得被进贡资格，根据对家的顺序判定：

- 二游：双贡
- 三游：单贡
- 末游：不进贡

当进贡一方拥有2张大王时可抗贡，即无需进贡。

## 出牌顺序

> 掼蛋中的出牌顺序包括：洗牌切牌抓牌、首牌、接风。
>
> 可随机挑选开局（第一局）方式。

顺时针抓牌，逆时针出牌。

洗牌->切牌->抓牌：头游上家洗牌，头游切牌，末游抓牌。

特别地，双下时头游下家抓牌。

![洗牌，切牌，抓牌](/images/img-table-game-guandan/上家洗牌-头游切牌-末游抓牌.png)

首牌：

- 单贡：进贡玩家获得
- 双贡时：进贡较大牌者获得
- 双贡且进贡牌值相等：头游下家

**接风**：当玩家出完手牌时，对手方不接牌，则对家获得出牌权。

## 报牌

> 报牌违规在比赛中会受到警告、扣分、停止一圈出牌权等，而在民间娱乐中一般无惩罚。

**报牌规则约定**

- 规则一：剩余10张牌主动报牌
- 规则二：剩余6张牌要主动报牌，剩余10张牌有问必报

报牌之后不允许继续追问。

## Q&A

### 1. 不同地区的规则有所不同，新手玩家如何选择？

掼蛋中的绝大多数规则是固定的，但在少数规则上有一定差异，往往体现为地域性，这一点与麻将类似。

因此，在开局前需要明确规则约定，可选**南京地区**的默认选项。


| 规则 | 约定                    | 默认 |
| ---- |-----------------------| ---- |
| 获胜 | 翻山3次失败则对手方获胜          | ✅   |
|      | 翻山打A直至获胜              |      |
| 报牌 | 剩余10张牌主动报牌            | ✅   |
|      | 剩余6张牌要主动报牌，剩余10张牌有问必报 |      |

### 2. 在翻山时，升级规则与普通级差异？

> 在双升中有庄家的概念，对手方抢庄需要消耗1级，而掼蛋仅在翻山时有类似概念。

普通局：升级与当前局主牌值无关。假如当前为对手方局，我方级数为5，本局对手方**双下**则下一局打8。

翻山局：当前级数为J、Q、K，则最大升级为A，下一局开始翻山。如果翻山回合失败，进行对手方主牌局，最快需要2次头游才能获胜，即一次头游获得翻山机会，一次头游且非末游翻山成功。
