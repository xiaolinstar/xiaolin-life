# 升级（双升），玩家攻略

江浙沪地区尤其是江苏人喜欢玩「掼蛋」。

![南京街头大爷掼蛋](/images/img-upgrade/guandan.jpg)

首次了解「掼蛋」的规则，让我想起来小时候玩的「双升」（也叫「升级」），同样好玩，具有较强的趣味性和智力博弈。在我看来，「掼蛋」就是「升级」基于「斗地主」规则的变体。

特别地，「升级」是花色为主顺序，而不是牌值。

四人桌游消遣娱乐不止是「麻将」和「掼蛋」，杀上几盘「升级」也是不错的选择。

我编写了「升级」规则手册，旨在提供清晰、逻辑严密的规则指引，无论是新手入坑还是老友切磋，都能随时通过以下章节快速翻看特定规则。

## 一、游戏配置

游戏配置：

- 玩家数：4名玩家，对家互为队友
- 牌局准备：2副标准扑克牌，共108张牌

![双福扑克牌](/images/img-upgrade/double-poke.jpg)

获胜机制：

1. 初始级数为2，率先打过A的一方获胜
2. 牌局分为「庄家方」和「抓分方」，主牌是「庄家方」的级数
3. 「庄家方」的目标是守擂，不让对方抢走「分数」，而「抓分方」的目标是抢走「分数」，每回合结束根据「分数」判定升级

简单来说，每一回合：

- **庄家方**：任务是保住手里的权，别让分数被对面抢走，顺便守好底牌。
- **抓分方**：任务是抢走「分数」，抢够 80 分就翻身农奴把歌唱，抢得越多升得越快！

## 二、基础规则

「升级」的基本规则与「掼蛋」和「斗地主」有2大区别：

- 「升级」以花色为主顺序，而不是牌值，初次接触不太容易接受。
- 「升级」是回合制游戏，通过得分来计算级数，而不是出完牌的顺序。

### 2.1 分值与级数

游戏中有三个数值的牌代表分数：

| 牌值 | 分值 |
| ---- | ---- |
| 5    | 5    |
| 10   | 10   |
| K    | 10   |

每一局结算的级数，由所获得分数决定，「抓分方」得分。

| 抓分方得分 | 下一局庄家 | 庄家方升级 | 抓分方升级 |
| ---------- | ---------- | ---------- | ---------- |
| 0          | 庄家方     | 3          | -          |
| (0, 40)    | 庄家方     | 2          | -          |
| [40, 80)   | 庄家方     | 1          | -          |
| [80, 120)  | 抓分方     | -          | 0          |
| [120, 160) | 抓分方     | -          | 1          |
| [160, 200) | 抓分方     | -          | 2          |
| 200        | 抓分方     | -          | 3          |

「抓分方」获得80分及以上，称为「上台」，被赶下台的「庄家」的顺位玩家为下一局「候选庄家」。

特别地，「抓分方」可能会得分超过200分，每多40分加一级。

### 2.2 牌力

牌力大小顺序，假如级数为10，黑桃♠️为主花色：

1. 大王
2. 小王
3. 主级牌：当前级数且主花色的牌（♠️10）
4. 副级牌：当前级数且副花色的牌（♥️♣️♦️10...）
5. 主牌：除主级牌外，主花色的牌（♠️A ♠️K ♠️Q...）
6. 副牌：除副级牌外，副花色的牌（♥️♣️♦️A ♥️♣️K ♥️♣️Q...）

在相同花色下，牌力大小顺序：A > K > Q > ... > 2。

如果2名玩家出相同的牌，先出牌者为大。

### 2.3 牌型

#### 基础牌型

基础牌型有2类：

- 对子：2张完全一样的牌，包括大小王对子
- 单牌：任意一张单牌

#### 拖拉机

连续的对子可构成「拖拉机」，例如：♥️55♥️66。「拖拉机」理论上可以任意长，最大的二级拖拉机为「大王对」。

特别地，主牌不能当作普通牌使用，例如：如果级数为6，♥️55♥️66♥️77不能组成三级的「拖拉机」，但是♥️55♥️77可以组成二级的「拖拉机」。

### 2.4 得分

每一回合中，牌局获胜方有得分权：

- 庄家方获胜：守分，不得分
- 抓分方获胜：得分，本回合中的分数牌（5、10、K）均为所得分数。

## 三、牌局流程

### 3.1 庄家与亮主

> 该规则存在多种版本，本文介绍最常见的玩法。

在摸牌结束后，剩下8张底牌，「庄家」亮级数牌，所亮牌花色为「主牌」（或主花色）。

如果「庄家」没有级数牌或选择不亮主，视为「让庄」，顺位玩家获得亮牌做庄权。

例如：

1. 玩家X为默认「庄家」，级数为5，那么亮牌♥️5，则♥️为主花色。
2. 玩家X没有级数牌5或选择不亮主，那么顺位玩家Y亮牌♠️6，则♠️为主花色。
3. 依次顺位亮牌，四位玩家均未亮牌，则该回合亮主失败，从底牌中随机抽取一张牌，构成对子者为「庄家」，所亮牌的花色为「主牌」。

**第一局**时双方都是2级，像打篮球一样抢主，率先亮主的玩家为「庄家」，之后每局「庄家」连庄或顺位做庄。

庄家扣底换牌：庄家获得8张底牌，然后再选择8张牌塞进底里。

小贴士❗️：把手里没用的杂牌或者分数藏进去。但千万小心，要是最后一张牌被对面赢了（抠底），那底里的分可就要翻倍送给人家了！

### 3.2 反主

当庄家回填好底牌后，其他玩家可以使用主牌对、小王对或大王对进行反主：

- 主牌对反主：反主后花色为主花色
- 大小王对反主：视为无主，即没有主花色

反主需要遵循强牌压弱牌原则，大王对 > 小王对 > 主牌对，因此主牌对可以反主一次，且必须在大小王反主之前。

反主也具有顺序性，庄家 -> 下一位 > ...，庄家可以自我反主，但无法使用报牌的主牌对反主，例如报牌的是♥️6，那么不能使用♥️66反主。

**每一次反主后，玩家可以获得扣底换牌特权。**

### 3.3 出牌（重要）

「庄家」获得首牌权，出牌为回合制，遵循以下规则：

1. 首领制：上一轮赢家获得本轮先手权。
2. 跟花原则：必须优先跟同花色；无同花色时可垫牌或毙牌。
3. 跟型原则：有对跟对。
4. 毙牌规则：用主牌杀掉副牌，张数与牌型必须完全匹配。

### 3.4 扣底判定

在最后一回合，如果「抓分方」获胜，则将8张底牌翻开得分：

- 单张：翻 1 倍
- 对子：翻 2 倍
- 拖拉机：翻 $2X$倍，$X$为拖拉机级数

### 3.5 分数结算

分数结算，然后计算级数，确定下一局的庄家，开始新的一局。

## 四、其他规则

### 4.1 甩牌

如果你觉得自己手里某一门花色的牌全是最大的，没对手了，你可以大喊一声“全甩了”！如果被别人抓到你手里还有小的，那可就要受罚。

若甩牌有误，需强制出其中最小的单张或对子。

### 4.2 报牌规则

存在另一种报牌版本，在摸牌结束后，使用3张牌报牌：

- 大王+红色对子（♥️♦️）
- 小王+黑色对子（♠️♣️）

只要「候选庄家」报牌，则「坐庄」成功。所有报牌的玩家翻开所报的3张牌，最大的玩家可以吸收掉其他玩家的牌，并且最大的牌花色为主花色。

吸牌之后，需要回牌，回牌的原则是，最小化扣底换牌的优势：

- 扣底换牌的玩家是失败方，则先摸底，回填8张底牌后，再回牌；
- 扣底换牌的玩家是成功方，则先回牌，再摸底。

回牌的规则是：不能给所吸收的牌的花色。

特别地，可以使用「拖拉机」进行报牌，并且大小优先级：拖拉机级数、拖拉机大小、对子大小

### 4.3 扣底规则

扣底的倍率在不同的规则中也有所不同：

- 单张：翻 2 倍
- 对子：翻 4 倍
- 拖拉机：翻 $4X$倍，$X$为拖拉机级数

## 五、棋牌技巧

帮队友拿回出牌权：看队友不行了，赶紧出个大对子接管比赛！

眉目传情（信号灯）：没大牌的时候，垫几张分给队友，或者故意出某种顺序的牌，让队友知道你手里啥情况。

先清理主牌：庄家要是主牌好，先出主对子把大家的防御都磨光，剩下的副牌就可以横着走啦！
