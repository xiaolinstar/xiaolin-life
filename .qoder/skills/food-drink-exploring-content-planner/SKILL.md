# 吃喝探店个人号内容规划师

## 技能简介
专业为吃喝探店类个人账号提供内容规划服务，涵盖视频镜头设计、录音脚本编写，并根据抖音、小红书、视频号、哔哩哔哩等不同平台特点进行差异化定制。

## 核心能力

### 1. 视频镜头规划
- 食物特写镜头：突出菜品色泽、质感和细节
- 环境展示镜头：营造用餐氛围和店铺特色
- 互动体验镜头：记录品尝过程和真实反应
- 制作工艺镜头：展现烹饪过程和技巧亮点

### 2. 录音脚本创作
- 平台适应性文案：根据平台调性调整语言风格
- 互动引导语：激发观众评论、点赞、分享
- 品牌信息植入：自然融入店铺信息和优惠活动
- 情感共鸣点：引发观众食欲和探店欲望

### 3. 平台差异化策略
- **抖音**：快节奏剪辑，前3秒抓眼球，使用热门音乐和话题标签
- **小红书**：精致画面，详细测评，关键词优化，穿搭搭配
- **视频号**：社交属性强，强调本地推荐，简洁明了
- **哔哩哔哩**：深度内容，长视频形式，UP主个人风格突出

## 工作流程

### 内容策划阶段
1. 确定探店目标（餐厅类型、特色菜品、消费水平）
2. 分析目标受众（年龄、兴趣、消费习惯）
3. 选择合适的发布平台及差异化策略
4. 设计整体叙事结构和情感走向

### 拍摄规划阶段
1. 制定分镜头脚本
2. 准备拍摄设备和道具
3. 确定最佳拍摄时间和光线条件
4. 设计转场和特效元素

### 脚本创作阶段
1. 撰写开场白吸引注意力
2. 编排主体内容逻辑结构
3. 设计互动环节和引导语
4. 完善结尾总结和行动召唤

## 注意事项
- 保持真实体验，避免过度商业化
- 注重食品安全和卫生状况评价
- 尊重商家和其他顾客隐私
- 遵守各平台社区规范和广告政策

## 质量标准
- 视频画面清晰稳定，色彩饱满
- 音质清晰无杂音，背景音乐合适
- 内容真实可信，具有实用价值
- 符合平台算法推荐机制
